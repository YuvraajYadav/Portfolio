I like understanding a system from the transistor up.

Projects

Selected work.

A processor built from scratch, and two embedded systems built to solve a real, specific problem.

16-bit Single-Cycle CPU

Jul 2025
Verilog HDL · AMD Vivado

Designed and built a custom 16-bit single-cycle CPU architecture from scratch — datapath, register file, ALU, instruction memory, data memory, and control unit. Defined a complete instruction set covering arithmetic, logic, load/store, branch, and jump operations, then verified the design through waveform analysis and testbench-driven simulation across all five stages: fetch, decode, execute, memory access, and write-back.

Outcome — a complete, independently verified CPU, with every instruction simulated and debugged stage-by-stage.

Smart Door Lock System

Aug 2024
Arduino IDE · TinkerCAD · MIT App Inventor

Built a password-protected, mobile-controlled door lock with live intrusion alerts. Designed and simulated the full circuit in TinkerCAD, then built a companion mobile app in MIT App Inventor for remote locking and monitoring.

Outcome — cut manual access by 60% and improved intrusion response time by 50%.

Multi-Zonal Appliance Control System

Sep 2024
Arduino IDE · Proteus Professional

Built a microcontroller-driven system that reads occupancy and environmental data from PIR motion sensors and DHT22 sensors across multiple zones, then drives relay-based switching to automatically manage appliances zone by zone.

Outcome — reduced power use in unoccupied zones while keeping conditions stable everywhere else.
